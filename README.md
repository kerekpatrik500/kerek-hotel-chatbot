# Kerék Hotel AI Chatbot

## Telepítés Replitra (5 perc)

### 1. Replit fiók
Regisztrálj a https://replit.com oldalon (ingyenes).

### 2. Új projekt létrehozása
- Kattints a "+ Create Repl" gombra
- Válaszd a **Node.js** sablont
- Adj nevet: `kerek-hotel-chatbot`

### 3. Fájlok feltöltése
Töltsd fel ezeket a fájlokat:
- `server.js`
- `package.json`
- `public/index.html`

### 4. API kulcs beállítása (FONTOS!)
A Replit bal oldalán kattints a 🔒 **Secrets** (zárbikon) menüre:
- Key: `ANTHROPIC_API_KEY`
- Value: az Anthropic API kulcsod (https://console.anthropic.com)

### 5. Indítás
Kattints a **Run** gombra. Megjelenik a publikus link, pl.:
`https://kerek-hotel-chatbot.felhasznalonev.repl.co`

## Testreszabás

A hotel adatait a `server.js` fájl tetején lévő `SYSTEM` változóban lehet módosítani.
